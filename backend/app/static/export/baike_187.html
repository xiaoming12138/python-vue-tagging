中国电影博物馆发展历程

编辑
中国电影博物馆(12张)
中国电影博物馆是经国务院批准，由国家广电总局和北京市政府共同建设的大型公共文化设施，是目前世界上最大的国家级电影专业博物馆，2007年2月10日正式对公众开放，2008年3月免费开放。凭借丰富的展陈、完善的设施、优质的服务、先进的技术，自对公众开放以来，观众已突破285万人次。馆内收藏电影拷贝、手稿、电影海报和电影器材等珍贵藏品3万多件。
中国电影博物馆常设展览《百年历程 世纪辉煌》展示了中国电影从1905年到2005年走过的百年历程，是目前我国乃至全世界第一次全景式、立体化、系统性呈现中国电影发展史和电影制作技术的大型综合展览。馆内还不定期举办临时展览。同时面向社会征集和电影有关的藏品　中国电影博物馆为观众配备了六个现代化电影厅，可同时放映70毫米胶片巨幕电影（IMAX）、数字电影和35毫米胶片电影。同步放映最新电影，精心挑选优秀国产经典影片公益放映，每月还举办主题展映活动。
业界荣誉(4张)
中国电影博物馆以“国家级、专业化、公益性、第一流”为定位，优质的服务，先进的设施，让这里成为业界人士进行集会、研讨和庆典活动的重要场所。“中国（北京）电影学术年会”、“全国少年儿童电影才艺展示活动暨少年儿童电影配音大赛”、“中国电影博物馆青年电影论坛”、“电影大讲堂”、“社会大课堂”、“公益影展观影团”、“电影音乐展示欣赏活动”、“中国电影博物馆之夜”等是中国电影博物馆开展社会教育，着力打造的系列化、品牌化专业传播活动。会员俱乐部受到影人影迷的欢迎，会员可以独享更加丰富的电影活动和更加优惠的观影。中国电影博物馆“影博网”和官方微博“影博汇”，为与观众“e”交流提供了平台。
馆内综合服务丰富多样，包括中央圆厅、多功能厅、贵宾厅、临时展览厅

，可以为电影界的集会、研讨和庆典活动提供场所。除此之外，还配有影吧·咖啡厅、快餐店、小型商店等配套设施。拥有一万平方米的免费停车场和充足的停车位。

中国电影博物馆建筑布局

编辑
中国电影从1905年开山之作《定军山》问世以来已经走过了一百多年的风雨历程。在中国电影的百年画卷里，有许多经典影片今天依旧闪烁着灿烂的光辉。在中国电影博物馆的展览区中，可以通过多种方式真实感受这些经典作品的魅力。
中国电影博物馆内设有20个展厅，其中序厅、1至10展厅是电影艺术展览厅，位于中国电影博物馆的二层和三层，以展示中国电影百年历程和电影艺术家的艺术成就为主。

中国电影博物馆展览区

展览区由“电影的发明”、“中国电影的诞生和早期发展”、 “革命战争时期的中国电影”、“新中国电影的创建与发展”、“改革开放新时期的中国电影”、“美术电影”、“儿童电影”、“科学教育电影、译制电影、新闻纪录电影”、“香港、澳门地区电影”、“台湾地区电影”10个展厅组成。

中国电影博物馆博览区

11-20 展厅为电影技术博览区，位于中国电影博物馆的四层，由“电影拍摄”、“电影美术”、“电影特殊摄影”、“传统电影特技”、“数字特技”、“电影录音”、“电影剪辑”、“电影洗印”、“电影动画”、“形形色色的电影”10个展厅组成。
电影技术博览区以展示电影制作技术和电影知识，揭示电影制作的奥秘为主要内容。您可以亲自动手做录音、拍短片，体验电影制作的乐趣，从拍摄、剪辑、美术、配音、配乐到特技、洗印等整个电影制作过程都有充分展示。观众参与性和互动性是博览区突出的特色。

中国电影博物馆影院区

中国电影博物馆内的影院区拥有全球最新的电影放映技术和设备，并设有多个放映厅，包括巨幕（IMAX）影厅和五个普通影厅（含 35毫米胶片影厅和数字影厅等），可同时容纳 1210位观众观影，其中包含 19个为残疾人提供的座位。
IMAX 影厅
巨幕影厅内
巨幕 ( IMAX )又称巨幅银幕，是目前世界上最先进的影像系统。IMAX 源自英文“Image maximum”（图像最大化）。中国电影博物馆内的巨幕影厅（IMAX 影厅 ) 庄重、大气，银幕宽 27米，高 21米，厅内设有 403个座位。该影厅引进了世界第一流的电影放映系统，使用的是 70毫米、15片孔的电影胶片，它的感光面积是普通胶片的 10倍，能容纳更多的信息，让观众感受到更丰富的视觉冲击力。巨幕 ( IMAX）画面的稳定性大大超过任何常规标准，可以放映二维和三维影片。巨幕 ( IMAX）多声道超级音响系统包含有超低音频道。专门为巨幕影厅设计的声源均衡音响系统，使观众在影厅内每个地方感受到的音量和音质是完全相同的。
拥有超大的银幕、更清晰的图像、更高的精密度、功能更强的放映设备以及更先进的多声道数字音效系统的巨幕影厅，带给观众强烈的视觉冲击，加之逼真的音响效果，远远超越了普通电影院的观影体验，被称为“电影的终极体验”。
4号影厅
4号影厅采用先进的数字放映机，可以播放宽银幕立体声电影和立体电影（三维影片），音效逼真、色彩动人，图像如水晶般清晰。本影厅具备完善的会议用音响，可以举办各类报告会、电影首映式等活动。
普通影厅
（35毫米胶片影厅）
豪华的35毫米胶片影厅，内饰精致、典雅，以放映国产优秀影片和国际精选影片为主，同时推出影人影迷座谈会，并邀请电影届资深人士为电影爱好者做佳片解析。

中国电影博物馆综合服务区

与八一电影制片厂合作拍摄电影《歼十出击》(9张)
中国电影博物馆的综合服务区包括中央圆厅（中央展厅）、多功能厅、贵宾厅、临时展览厅，可以为电影界的集会、研讨和庆典活动提供场所，先进的同声传译设备保证了国际交流的顺利进行。中国电影博物馆将逐步成为集电影艺术展览、电影文化教育和电影艺术交流为一体的大规模、综合性文化设施。
除此之外，中国电影博物馆还配备有咖啡厅、快餐店、小型商店等配套设施，让观众徜徉于光影艺术的同时，体会到无微不至的关怀。

中国电影博物馆公益影展

中国电影博物馆常年放映优秀国产影片,并定期举办主题公益影展。今年已成功举办过反腐倡廉主题影展、大学生电影节获奖作品展、“八一礼赞”军事题材影片展等等；并随时关注影人动态推出主题展映，如“追忆一代大师——陈强”电影公益展映、“新中国人民演员——二十二大影星”影片展、直面社会现实，传递人文关怀——馆藏谢晓东现实题材电影作品展映等。中国电影博物馆还为迎接党的十八大胜利召开，推出了“喜迎党的十八大”公益电影展映活动。
中国（北京）电影学术年会
中国（北京）电影学术年会(18张)
为充分发挥中国电影博物馆的学术科研职能，促进前沿电影学术观点的生发、碰撞、交流、切磋，为电影业界、学界搭建沟通交流平台，自2009年起，中国电影博物馆每年举办一届学术年会。2011年起， 在上级领导的亲切关怀下，年会更名为“中国（北京）电影学术年会”。年会得到了中共北京市委宣传部、国家广电总局电影局的有力指导和北京市广播电影电视局的大力支持。
学术年会关注当下热点，已举办了四届分别以“新中国电影与观

《影博·影响》影评人俱乐部活动
《影博·影响》影评人俱乐部活动(9张)
《影博·影响》影评人俱乐部活动是中国电影博物馆馆刊《影博·影响》创办的公益性观影评论活动，旨在引导电影评论健康发展。自2009年创办以来，每月举办1—2次活动，结合上映的国内外重点影片，影评人俱乐部组织影评人观看和评论，整理发言集萃。同时，针对影评人对该片各项指标的评价等内容开展问卷调查，并和普通观众的打分情况对比研究，将发言集萃和影片问卷调查报告同时刊发在当期《影博·影响》（中国电影博物馆馆刊）上，供业界、学界参考。主要特色：一是活动的时效性。影评人俱乐部每月都会结合重点影片开展1—2次活动。二是内容的广泛性。在影片选择上，既关注国产主流重点影片，又关注中小成本影片和艺术性较强的影片。三是影评人代表性强，在年龄、职业、收入等方面构成多样。有中青年观众，也有退休老人；有专业学生、电影发烧友，也有就是单纯热爱电影的普通观众。
中国电影博物馆青年电影论坛
中国电影博物馆青年电影论坛(15张)
中国电影博物馆致力于在电影界和观众之间搭建沟通的桥梁，于2007年起以每年举办“青年电影论坛”的方式，助推中国青年电影人作品的影响和培养青年电影观众对中国电影的热爱，至今已成功举办了六届。青年电影论坛的举办得到了国家广电总局电影局和北京市委宣传部、北京市广播电影电视局的指导和中国电影人的大力支持，在中国电影界、电影观众和社会各界产生了良好影响和口碑，逐渐成长为在社会上有一定影响力的电影文化品牌活动。
国际交流(31张)
已经举办的六届青年电影论坛分别是：2007年第一届青年电影导演论坛、2008年第二届青年电影编剧论坛、2009年第三届青年电影发行人论坛、2010年第四届青年电影特技特效制作人论坛、2011年第五届青年电影音乐人论坛和2012年第六届青年电影美术人论坛。六届青年电影论坛共有中国（含香港、台湾、澳门）、美国、日本、韩国等国内外电影导演、电影编剧、电影发行人、电影特技特效人、电影音乐人和美电影术人等百余位青年电影精英、电影界前辈和电影管理机构领导莅临论坛演讲，数千位电影从业者、电影文化艺术专业学生、热爱电影的影迷聆听了演讲、大课堂和座谈会，观摩了展览和展映，专家、学者、电影主创和观众进行了热烈互动。在论坛上，青年影人、青年听众是主角，畅所欲言，气氛活跃，内容丰富，形式活泼，电影特色鲜明。六年间，中国电影博物馆“青年电影论坛”也逐渐形成了自己的风格特色，即：紧扣电影行业主题，关注青年发展前沿，着力解构最新成果，影响观众热爱电影。
目前，中国电影博物馆正在筹备2013年第七届青年电影论坛，内容拟定为“青年电影化妆师论坛”。并且继续推动和争取与业界机构、品牌活动、优势资源的合作，持续锻造打磨，将中国电影博物馆青年电影论坛的影响力继续提升，为建设首都文化中心、为中国电影大发展、大繁荣贡献力量。
中国电影博物馆电影音乐展示欣赏活动
中国电影博物馆电影音乐展示欣赏活动(10张)
中国电影博物馆“电影音乐展示欣赏”活动自2009年3月推出以来，至今已经举办过近百场演出。活动通过丰富多彩的形式，为观众展现中外优秀电影中经典音乐的独特魅力，用经典电影音乐的魅力感染人、打动人、引导人，同时使观众在欣赏电影音乐后，能够更多的了解有关音乐、电影音乐以及电影音乐背后的故事，收获更加丰富的音乐和电影方面的知识，营造浓厚的电影艺术氛围。受到了观众的一致好评和喜爱。

中国电影博物馆公益影展

公益影展观影团(9张)
中国电影博物馆“公益影展观影团” 自2009年2月24日启动。每次活动都是从公益影展展映的优秀影片里，挑选一部经典之作组织“回看”观摩并撰写影评。目前本活动主要组织高校大学生团体观看我馆举办的不同主题的公益影展中的影片，进行有针对性的电影文化传播。我们也计划将本活动推向更多的社会团体，打造一个聚电影主创、影视专家、广大观众于一堂的，集观影、座谈、影评于一体的，深入探索影片，交流心得，传播电影文化的活动。让这些经典影片不因时光而消逝，任由思想和情感在影海中徜徉。
会员俱乐部
中国电影博物馆会员俱乐部成立于2007年，以公益性服务为宗旨，以专业化服务为定位，面向喜爱电影、尊重知识、热心博物馆事业的各界人士提供优越和优惠的服务，是实现中国电影博物馆传播电影文化和社会教育的功能的有效载体，是影迷定期相聚和交流体验电影文化的平台。
会员俱乐部(10张)
中国电影博物馆会员俱乐部成立以来，注重提升服务品质，满足影迷多样化需求。通过组织系列化的观摩沙龙、电影拍摄讲座、青少年动漫知识体验、公益放映、爱心之旅等系列化的电影文化活动，加强交流与互动，以丰富多样的活动内容吸引了众多影迷成为会员。
电影文化进社区，经典电影大家看
中国电影博物馆作为专业电影博物馆，非常重视发挥博物馆公益性职能，用电影放映的独特方式，深入基层服务群众，深入传播电影知识，丰富观众的文化生活，我馆自2008年起就开展了电影文化进社区、进乡村、进学校系列活动，所到之处受到了大家的积极参与和热烈欢迎。
电影文化进社区，经典电影大家看(10张)
电影文化进社区活动是我馆持续开展的公益服务活动，几年来，走入近百个社区、乡村、学校、敬老院和企事业单位，为社区居民、学生、敬老院的老人和企事业职工们放映电影，活动覆盖了北京市朝阳区、东城区、西城区、海淀区、丰台区、昌平区、石景山区、大兴区、顺义区、通州区等区县，还有延庆、房山、密云等远郊区县和河北省三坡镇等乡村，通过面向青少年的电影动漫体验、经典国产影片放映、优秀电影美术获奖影片放映、电影知识问答等主题内容，让观众欣赏精彩电影、学习电影知识，感受电影文化。
今后，中国电影博物馆还将进一步延续此项品牌公益活动，继续用电影巡走的方式发挥优秀电影作品感染人、教育人的职能，扩大受众人群和覆盖范围，将公益进行到底。

中国电影博物馆露天电影

中国电影博物馆露天电影放映(10张)
为营造良好的社会氛围，坚持文化惠民，中国电影博物馆每年中秋节前后举办露天电影招待专场活动，精选观众喜闻乐见的不同类型的影片为我馆驻地朝阳区崔各庄地区居民露天放映。丰富和活跃当地居民的精神文化生活，传播电影文化，推进城乡文化活动热潮。
中国电影博物馆“电影大讲堂”活动介绍
“电影大讲堂”是中国电影博物馆开展的公益性电影文化系列讲座，邀请电影、文化学界与业界高端人士作为主讲人，以讲座、赏析交流为主要形式，讲述电影文化、电影艺术、电影历史、电影科技等内容，展示、交流、赏析优秀影片，传播中国电影优
中国电影博物馆“电影大讲堂” 活动(8张)
秀文化，坚持公益性、专业性、前沿性和高质量。在这里，观众能与电影学者、专家交流对电影的热爱，与电影导演、演员等主创人员分享观影感受，讨论电影话题，了解台前幕后不为人知的感人故事。“电影大讲堂”专题讲座还与社会热点相结合，注重知识性、趣味性，注重观众参与和实际效果。参与观众既可以领略、学习电影专业知识与电影艺术文化内容，也可以感受观看、赏析、交流优秀影片带来的美好体验。
中国电影博物馆“社会大课堂-电影课堂”
中国电影博物馆是北京市教委2008年首批授予“北京市社会大课堂中小学课程教学活动实验基地”称号的资源单位之一，“社会大课堂-电影课堂”是我馆面向中小学生群体的社会教育系列活动的统称。该活动旨在利用馆内丰富的电影资源优势，以电影文化艺术教育和电影科普教育为主要内容，通过电影教学课程实践、主题活动、观影交流、参观体验等教学手段和方式提高中小学生的科学文化素质，为中小学生提供丰富多彩的校外活动。受众群立足于首都中小学生，同时，向北京市以外的地区扩展。
中国电影博物馆“社会大课堂-电影课堂”(30张)
“社会大课堂-电影课堂”活动课程设置将中国电影博物馆展陈内容与中小学课本教学内容相结合，开展课题研究、参观体验、观影交流等活动。现有课程包括《电影特技解密》、《动画电影》、《电影中的化学》、《电影中的舞蹈》、《电影中的历史真相》、《电影中的色彩》、《电影中的音乐》等。另外，“佳片赏析影人交流活动”、“找宝行动”、“我的动画我来做”、走进山区和打工子弟校园，以及以电影为载体手段，结合雷锋日、传统节日等开展系列主题教育活动，也是中小学生学习电影文化科技艺术知识的好机会。
庆祝新中国成立60周年电影系列活动
庆祝新中国成立60周年电影系列活动展览(9张)
为庆祝中华人民共和国成立60周年，回忆新中国电影60年走过的足迹和见证60年来取得的成就与辉煌，由国家广电总局电影局和中共北京市委宣传部主办，北京市广播电影电视局和北京市文物局支持，中国电影博物馆承办，中国电影集团公司、上海电影集团公司、长影集团和八一电影制片厂四大国有电影制片机构共同协办的《前进！前进！前进！进！——庆祝新中国成立60周年电影系列活动》自2009年7月31日在中国电影博物馆拉开帷幕，历时四个月，于11月30日圆满闭幕。《前进！前进！前进！进！——庆祝新中国成立60周年电影系列活动》由7项内容组成，即：“新中国电影六十年国产影片成就展”、“光影60年——新中国优秀电影作品展映”、“新中国电影六十年国产影片剧照展”、“新中国电影六十年国产影片海报展”、“《对话·博物馆》特别节目——‘电影人生’座谈会”、“‘前进！前进！前进！进！——新中国电影与观众’问卷调查活动”和“中国电影博物馆2009年学术年会”
全国少年儿童电影才艺展示活动
全国少年儿童电影才艺展示活动(14张)
为弘扬社会主义核心价值观，丰富少年儿童的课余生活，发挥首都全国文化中心的引领作用，自2010年起，中国电影博物馆与中国关心下一代工作委员会办公室合作，在全国范围内共同举办了全国少年儿童电影才艺展示活动暨少年儿童电影配音大赛，该活动每年4月启动至8月结束，历时近4个月，面向全国的少年儿童免费参与，得到众多社会公益性机构的支持和协办，邀请众多老艺术家和电影业界专家参与和指导。活动以“关注少儿成长、启迪少儿心智、丰富少儿生活、繁荣少儿电影”为宗旨，为少年儿童提供学习光影知识和展示电影才华的平台，让更多的少年儿童走进中国电影世界，让优秀国产儿童电影的银幕形象深植少年儿童心中，发挥优秀电影艺术作品感染人、塑造人的作用，启迪孩子们对艺术的向往和对高尚情操的追求，让他们在幼小的心灵中树立社会主义核心价值观。

2012年与前两届相比进一步扩大了规模，丰富了参与内容，吸引了全国29个省市自治区近50万名少年儿童和家长的关注，有6万余名小选手参加了各赛区的初赛、复赛以及相关的电影主题活动。第十届全国人大常委会副委员长、中国关工委主任顾秀莲亲自参加颁奖仪式并讲话，对本次大赛给予了高度评内容价。
第四届少年儿童电影配音大赛暨京港澳台少年儿童电影夏令营落幕，中国电影博物馆被授予“中国关心下一代工作委员会青少年影视教育基地”
